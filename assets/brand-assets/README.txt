STRAKK — BRAND ASSET PACK
=========================

The mark: three forward-leaning speed lines closed by a solid dot ("///.").
Wordmark: "Strakk" set in Archivo Extra-Bold (800).

------------------------------------------------------------
COLORS
------------------------------------------------------------
Icon gradient   : #FF8A2B  ->  #FF5E00   (linear, ~150deg / top-left to bottom-right)
Accent (flat)   : #FF6A12
Ink / dark bg   : #0A0A0D
Mark on icon    : #FFFFFF (with 50% / 78% / 100% opacity steps on the three lines)

Corner radius (app icon) : 22.37% of size (iOS "squircle" ratio)

------------------------------------------------------------
FONT
------------------------------------------------------------
Archivo — https://fonts.google.com/specimen/Archivo
Wordmark weight: 800 (Extra-Bold). Free, open-source (OFL).

------------------------------------------------------------
FOLDERS
------------------------------------------------------------
svg/        Master vectors (scale to any size, never blurry)
              strakk-icon.svg         full app icon (gradient squircle + mark)
              strakk-glyph-white.svg  white mark only, transparent
              strakk-glyph-orange.svg orange mark only, transparent

app-icon/   iOS / Apple App Store
              appstore-1024.png       1024 square, full-bleed (Apple rounds it)
              ios-1024..40.png        rounded, ready-to-use previews

play-store/ Google Play / Android
              playstore-512.png       512 square store listing icon
              android-192 / 512.png   launcher sizes (full-bleed)
              android-maskable-512.png maskable (safe-area) for adaptive icons

web/        Website + PWA
              favicon-16/32/48/64.png
              apple-touch-icon-180.png
              icon-192.png / icon-512.png  (PWA manifest)

glyph/      Transparent mark only (in-app, watermarks, social avatars)
              strakk-mark-white-1024.png
              strakk-mark-orange-1024.png

lockup/     Horizontal logo + wordmark
              strakk-lockup-light.png  dark wordmark on light bg
              strakk-lockup-dark.png   white wordmark on dark bg

------------------------------------------------------------
QUICK USE
------------------------------------------------------------
iOS app icon ........ app-icon/appstore-1024.png (no transparency, no rounding)
Play Store icon ..... play-store/playstore-512.png
Android adaptive .... play-store/android-maskable-512.png (foreground)
Website favicon ..... web/favicon-32.png  +  web/apple-touch-icon-180.png
In-app header ....... lockup/strakk-lockup-dark.png  (or the SVG glyph)
Anywhere, any size .. svg/  (preferred — infinitely scalable)

Need other sizes, an SVG lockup, or a monochrome black/white set? Just ask.
